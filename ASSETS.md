## 💡 使用示例

### Markdown 引用图片
```markdown
![Kenna 头像](https://cdn.jsdelivr.net/gh/Liz7789/yaruresume@main/assets/images/02-avatar-sunshine.jpg)
```

### HTML 引用
```html
<img src="https://cdn.jsdelivr.net/gh/Liz7789/yaruresume@main/assets/images/02-avatar-sunshine.jpg" 
     alt="Kenna 头像" 
     style="width: 200px; border-radius: 50%;" />
```

### 飞书文档
直接粘贴 jsDelivr 链接，飞书会自动渲染为图片。
